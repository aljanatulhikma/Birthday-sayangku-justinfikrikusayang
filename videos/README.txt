Taruh video-video kamu di folder ini.

Cara termudah: upload video apa adanya (nama bebas), lalu tulis nama filenya
satu per baris di file "videos.txt" yang ada di folder ini juga.

Kalau belum ada video yang mau dipakai, folder ini boleh dikosongkan,
website tetap berjalan normal, bagian "Video Kenangan" otomatis tidak
menampilkan apa-apa (tidak ada error).

Saran: pakai video pendek-pendek (di bawah 25-30MB per video) supaya
tidak lama loading pas dibuka orang lain. Format sebaiknya .mp4.
